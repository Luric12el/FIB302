# FIB302

## Descrição
Este repositório é dedicado às atividades e projetos da disciplina Pesquisa em Física II (FIB302). Aqui estarão contidos códigos básicos de Python e Wolfram Mathematica, de um estudante em processo de aprendizado, com intuito meramente educacional, visando melhor entender algumas das ferramentas usadas na física.
